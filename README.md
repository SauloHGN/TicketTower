### ⚙️ Prerequisites

• Node.js 20.18+ <br>
• Angular 17.2.2+ <br>
• Nest 10.0.0+

### 🚀 Instalation

_Front-end_

```
npm install -g @angular/cli
```

```
npm install dependencies
```

```
ng serve
```

`http://localhost:4200/`

_Back-end_

```
cd backend
```

```
npm i -g @nestjs/cli
```

```
nest start
```

`http://localhost:3000/`
